# 🎮 게임 내 AI 모드 명령어 사용법

## 📋 기본 정보

### **명령어 기본 형식**
```
/modpackai <서브명령어> [매개변수]
```

### **권한 설정**
- **기본 사용자**: `modpackai.use` (모든 플레이어에게 기본 허용)
- **관리자**: `modpackai.admin` (OP 권한 필요)

---

## 🗣️ AI 채팅 명령어

### `/modpackai chat`
**설명**: AI 어시스턴트와 대화할 수 있는 GUI 창을 엽니다.

**사용법**:
```
/modpackai chat
```

**기능**:
- 📝 채팅 입력창에서 질문 작성
- 💬 이전 대화 기록 확인
- 🎯 모드팩 관련 전문 답변
- 🔄 실시간 AI 응답

**예시 질문**:
- "다이아몬드 검 만드는 법 알려줘"
- "Create 모드의 기계는 어떻게 작동해?"
- "철 블록으로 뭘 만들 수 있어?"
- "이 모드팩에서 가장 강력한 무기는 뭐야?"

---

## 🛠️ 제작법 조회 명령어

### `/modpackai recipe <아이템명>`
**설명**: 특정 아이템의 3x3 제작법을 GUI로 표시합니다.

**사용법**:
```
/modpackai recipe <아이템명>
```

**매개변수**:
- `<아이템명>`: 제작법을 찾을 아이템의 이름 (한글/영어 모두 가능)

**예시**:
```
/modpackai recipe 다이아몬드 검
/modpackai recipe Diamond Sword
/modpackai recipe 철 블록
/modpackai recipe Iron Block
/modpackai recipe 골든 사과
/modpackai recipe Golden Apple
```

**표시 정보**:
- 🎯 3x3 제작 그리드
- 📦 필요한 재료 목록
- 🏷️ 결과 아이템 정보
- 📋 제작법 타입 (shaped/shapeless)

---

## 🤖 AI 모델 관리 명령어

### `/modpackai models`
**설명**: 사용할 AI 모델을 선택할 수 있는 GUI를 엽니다.

**사용법**:
```
/modpackai models
```

**선택 가능한 모델**:
- **GPT-3.5 Turbo** (OpenAI) - 빠르고 저렴
- **GPT-4** (OpenAI) - 정확도 높음
- **Claude 3 Haiku** (Anthropic) - 빠르고 효율적
- **Claude 3 Sonnet** (Anthropic) - 균형잡힌 성능
- **Gemini Pro** (Google) - 무료 크레딧 제공

**특징**:
- 💰 무료 크레딧 소진 시 자동 알림
- 🔄 실시간 모델 전환
- 📊 각 모델별 성능 정보

### `/modpackai current`
**설명**: 현재 사용 중인 AI 모델의 정보를 표시합니다.

**사용법**:
```
/modpackai current
```

**표시 정보**:
- 🤖 모델 이름
- 🏢 제공 업체
- 💰 요금제 (무료/유료)
- 📝 모델 설명

## 🔧 관리자 명령어

### `/modpackai switch <모드팩명> [버전]`
**설명**: 새로운 모드팩으로 변경합니다. (관리자 전용)

**사용법**:
```
/modpackai switch <모드팩명> [버전]
```

**매개변수**:
- `<모드팩명>`: 변경할 모드팩의 이름 (필수)
- `[버전]`: 모드팩 버전 (선택사항, 기본값: 1.0)

**예시**:
```
/modpackai switch CreateModpack
/modpackai switch FTBRevelation 1.0.0
/modpackai switch AllTheMods 1.19.2
```

**자동 처리 과정**:
1. 🔍 새 모드팩 파일 분석
2. 💾 데이터베이스 업데이트
3. 🌐 언어 매핑 재생성
4. 🔄 RAG 벡터 데이터 업데이트
5. ⚙️ 설정 파일 업데이트

**주의사항**:
- ❗ 관리자 권한(`modpackai.admin`) 필요
- 📁 새 모드팩 파일이 서버에 업로드되어 있어야 함
- ⏱️ 처리 시간이 소요될 수 있음

## 🚨 문제 해결

### **명령어가 작동하지 않을 때**
1. **권한 확인**: `/op <플레이어명>` (관리자 명령어용)
2. **플러그인 상태**: `/plugins`로 ModpackAI 확인
3. **서버 재시작**: 플러그인 재로드

### **AI 응답이 없을 때**
1. **API 키 확인**: 서버 관리자에게 문의
2. **모델 변경**: `/modpackai models`로 다른 모델 선택
3. **네트워크 연결**: 서버 인터넷 연결 상태 확인

### **제작법이 표시되지 않을 때**
1. **아이템명 확인**: 정확한 이름으로 재시도
2. **모드팩 확인**: 해당 아이템이 현재 모드팩에 포함되어 있는지 확인
3. **한글 변환**: 영어 이름으로 재시도

### **GUI가 열리지 않을 때**
1. **아이템 확인**: AI 어시스턴트 아이템을 들고 있는지 확인
2. **우클릭**: 정확히 우클릭했는지 확인
3. **권한 확인**: `modpackai.use` 권한이 있는지 확인
