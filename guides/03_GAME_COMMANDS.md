# 🎮 게임 내 AI 모드 명령어 사용법

## 📋 개요

이 가이드는 마인크래프트 게임 내에서 AI 어시스턴트를 사용하는 모든 방법을 설명합니다.
AI 어시스턴트는 모드팩 관련 질문에 답하고, 제작법을 제공하며, 게임 플레이를 도와줍니다.

## 🗣️ 기본 명령어

### 1. `/ai <질문>` - 바로 질문하기

가장 간단한 방법입니다. AI에게 바로 질문을 할 수 있습니다.

**사용법**:
```
/ai 철 블록은 어떻게 만들어?
/ai 다이아몬드는 어디서 찾을 수 있어?
/ai Applied Energistics 2 시작하는 방법
/ai 자동화 농장 만들기
```

**예시 대화**:
```
플레이어: /ai 네더라이트 장비 만드는 방법
AI: 네더라이트 장비를 만들려면 다음 단계를 따르세요:

1. 네더라이트 스크랩 4개 + 금 주괴 4개 = 네더라이트 주괴 1개
2. 다이아몬드 장비 + 네더라이트 주괴 = 네더라이트 장비

네더라이트 스크랩은 고대 잔해(Ancient Debris)를 제련하면 얻을 수 있습니다.
고대 잔해는 Y=8-22 레벨에서 가장 많이 발견됩니다.
```

### 2. `/modpackai` - 상세 명령어들

다양한 기능을 세부적으로 사용할 수 있습니다.

#### **기본 사용**

| 명령어 | 설명 | 예시 |
|--------|------|------|
| `/modpackai help` | 도움말 보기 | `/modpackai help` |
| `/modpackai chat` | AI 채팅 GUI 열기 | `/modpackai chat` |
| `/modpackai current` | 현재 AI 모델 정보 | `/modpackai current` |

#### **제작법 조회**

| 명령어 | 설명 | 예시 |
|--------|------|------|
| `/modpackai recipe <아이템>` | 제작법 GUI로 보기 | `/modpackai recipe 다이아몬드` |
| `/modpackai recipe <영어명>` | 영어 아이템명 사용 | `/modpackai recipe diamond` |

#### **모델 관리 (선택사항)**

| 명령어 | 설명 | 예시 |
|--------|------|------|
| `/modpackai models` | 사용 가능한 AI 모델 목록 | `/modpackai models` |
| `/modpackai switch <모델>` | AI 모델 변경 | `/modpackai switch openai` |

## 📖 AI 어시스턴트 아이템

### 1. 아이템 정보

**아이템**: 책 (Book)
**이름**: §6§l모드팩 AI 어시스턴트
**설명**:
- 우클릭하여 AI와 대화하세요
- 모드팩 관련 질문에 답변해드립니다

### 2. 획득 방법

**관리자 명령어**:
```
/give @p book 1
/give <플레이어명> book 1
```

**일반 플레이어**: 관리자에게 요청하거나 지급받기

### 3. 사용법

1. 📖 AI 어시스턴트 아이템을 손에 들기
2. 🖱️ 우클릭하여 AI 채팅 GUI 열기
3. 💬 질문하기
4. ✅ AI 답변 확인

## 🎯 AI 모델 선택

### 1. 사용 가능한 모델들

**Gemini 2.5 Pro** (기본, 권장):
- 🌐 웹검색 지원으로 최신 정보 제공
- 💰 GCP 크레딧으로 무료 사용 가능
- 🎯 높은 정확도

**GPT (OpenAI)** (백업):
- ⚡ 빠른 응답 속도
- 📚 풍부한 지식 베이스
- 💵 API 사용료 필요

**Claude (Anthropic)** (백업):
- 🤖 자연스러운 대화
- 📝 상세한 설명
- 💵 API 사용료 필요

### 2. 모델 전환

**GUI에서**:
1. AI 채팅 창 열기
2. "모델 선택" 버튼 클릭
3. 원하는 모델 선택

**명령어로**:
```
/modpackai models        # 사용 가능한 모델 목록 확인
/modpackai switch gemini # Gemini로 변경
/modpackai switch openai # OpenAI로 변경
/modpackai switch claude # Claude로 변경
```

## 🚨 문제 해결

### 1. 명령어가 작동하지 않을 때

**확인사항**:
- ✅ 플러그인이 정상 로드되었는지 확인
- ✅ 권한 설정 확인 (`modpackai.use`)
- ✅ 서버 콘솔 에러 메시지 확인

**해결 방법**:
```
# 서버 관리자 권한으로
/reload confirm
/modpackai help  # 플러그인 작동 테스트
```

### 2. AI가 응답하지 않을 때

**확인사항**:
- ✅ 백엔드 서비스 실행 상태
- ✅ API 키 설정 확인
- ✅ 네트워크 연결 상태

**테스트 방법**:
```
/modpackai current  # 현재 모델 상태 확인
/modpackai models   # 사용 가능한 모델 확인
```

### 3. GUI가 열리지 않을 때

**해결 방법**:
1. 인벤토리 공간 확보
2. 다른 GUI 창 닫기
3. 서버 재접속
4. `/modpackai chat` 명령어로 시도

## 📚 고급 사용 팁

### 1. 효과적인 질문법

**좋은 질문**:
```
❌ "뭐 해야 해?"
✅ "Create 모드로 자동 농장 만들고 싶은데 어떻게 시작해야 해?"

❌ "이거 뭐야?"
✅ "Applied Energistics 2의 ME 드라이브 사용법을 알려줘"
```

### 2. 한글/영어 혼용

AI는 한글과 영어 아이템명을 모두 이해합니다:
```
/ai 다이아몬드 검 만들기      # 한글
/ai diamond sword recipe     # 영어
/ai Diamond Sword 제작법     # 혼용
```

### 3. 단계별 가이드 요청

복잡한 작업은 단계별로 요청하세요:
```
/ai AE2 시작 가이드를 단계별로 알려줘
/ai Create 모드 기초부터 고급까지 순서대로 설명해줘
```
